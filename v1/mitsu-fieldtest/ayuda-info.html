Informacion de la orden

Volumen mínimo de agua en el circuito primario

Este volumen se solicita para que cuando la unidad entre en proceso de desescarche tenga un volumen mínimo contra el que trabajar, la ausencia de esta volumen puede producir daños en el intercambiador de placas.

Vaso de expansión del circuito primario

Este elemento se solicita para absorber las dilataciones del circuito de producción, en caso de no tener vaso de expansión o que este no esté suficientemente dimensionado para el volumen del circuito, el circuito no sería capaz de absorber esta dilataciones y saltaría la válvula de seguridad.

Vaso de expansión para ACS

Este elemento se solicita para absorber las dilataciones del circuito de consumo, en caso de no tener vaso de expansión o que este no esté suficientemente dimensionado para el volumen del circuito, el circuito no sería capaz de absorber esta dilataciones y saltaría la válvula de seguridad.

Instalación del tubo de desagüe

Es necesario la conducción del desagüe para evitar las fugas de condensados.
